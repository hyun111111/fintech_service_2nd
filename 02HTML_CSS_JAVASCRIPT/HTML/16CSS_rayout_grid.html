그리드 레이아웃

그리드 레이아웃은 웹페이지에서 2차원(행,열)을 이용해 레이아웃을 설계하는 방식

플렉스 방식보다 더 정교하게 레이아웃 설계 가능

행(row)
그리드에서 레이이아웃에서 가로줄을 의미
열(COLUMN)
그리드에서 레이이아웃에서 세로줄을 의미
그리드 셀(grid-cell)
행과 열이 만나 이루어지는 하나의 공간
그리드 갭(grid-gab)
그리드 셀과 그리드 셀 사이의 간격
그리드아이템(grid-item)
그리드 셀 안에서 표현되는 컨텐츠
그리드 라인(grid-line)
그리드 행과 열을 그리는 선
그리드 넘버(grid-number
그리드 라인에 붙는 번호
그리드 컨테이너(grid-container)
그리드 레이아웃의 전체 내용을 담고 있는 최상위 부모요소
